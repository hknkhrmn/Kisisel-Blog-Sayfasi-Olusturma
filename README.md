# Blog Projesi

Bu proje basit bir blog sayfasıdır. HTML5 kullanılarak yapılmıştır.

## Proje Hakkında

Bu blogda 3 farklı konu bulunmaktadır:
- Sınırlarımız
- NBA takımı Rockets'ın ligdeki durumu
- Kariyer değişimi

## Kullanılan Teknolojiler

- HTML5
- Temel HTML etiketleri

## Özellikler

- Ana başlık ve alt başlıklar
- Her yazıda görseller var
- Listeler (sıralı ve sırasız)
- Dış linklere bağlantılar
- Basit ve anlaşılır tasarım

## Nasıl Çalıştırılır

1. index.html dosyasını indir
2. Dosyaya çift tıkla
3. Tarayıcıda açılacak

## Proje Yapısı

- Her blog yazısında başlık, görsel, paragraf ve liste var
- Toplam 2 dış link kullanıldı
